# 復能小廚房「番茄蛋任務」候選臨床規格

> 狀態：**待仲暘人工確認**。本文件只記錄產品互動候選，不代表正式醫療主張、處方或療效承諾。

## 關卡目的

- 將原本單一揮手碰目標的玩法，改成抓取搬運、穩定保持、畫圈攪拌三段任務。
- 成功條件是完成流程，不以速度、分數、準確率或排行榜判定。
- 滑鼠與觸控為無鏡頭時的產品體驗入口；鏡頭模式以食指位置與拇指／食指捏合操作。

## 待人工確認的候選設定

| 項目 | 目前候選值 | 待確認問題 |
|---|---:|---|
| 食材數量 | 3 個，固定順序 | 是否需要依個案能力減為 1–2 個或允許治療師設定？ |
| 抓取半徑 | 畫面正規化約 0.14 | 對動作控制較弱者是否足夠寬容？ |
| 穩定保持 | 1.5 秒 | 是否適合做為預設值，是否需提供更短選項？ |
| 畫圈範圍 | 鍋中心半徑 0.13–0.38 | 是否需依校正後活動範圍縮放？ |
| 畫圈次數 | 3 圈，不限時間 | 是否需提供 1／2／3 圈處方選項？ |
| 中斷處理 | 當步重來，不倒扣、不計失敗 | 是否符合臨床端希望的提示方式？ |

## 本輪未建立的主張

- 不宣稱此關卡改善特定關節角度、肌力、協調能力或日常功能。
- 不自行設定疼痛、疲勞、活動角度或禁忌門檻。
- 未經人工簽核前，不將候選設定描述為治療建議或對個案主動推廣。
