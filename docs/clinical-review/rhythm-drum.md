# 臨床審核 — 節奏復能鼓（rhythm-drum）

> 狀態：**草稿，待臨床顧問（仲暘）簽核**。簽核前僅內部測試、不主動推廣給個案。

## 規格卡

```yaml
game_id: rhythm-drum
name: 節奏復能鼓
category: neuro
clinical_goal: 上肢雙側交替揮擊、節奏性動作（節拍聽覺＋視覺提示之動作計時訓練）、hard 難度以中央鼓位誘導跨中線動作
target_population: 中風上肢恢復期（可主動觸及左右前方目標者，約 Brunnstrom IV 以上）；長者上肢協調與節奏訓練
mechanics: 霓虹音符沿光軌落向左/右（hard 加中央）鼓面，音符抵達鼓面瞬間伸手觸擊命中；左右輪流出拍誘導雙手交替；60 秒/場
detection: MediaPipe Hand 腕點（useMoleDetector 既有邏輯）；音符拍點 ±goodMs 開啟可擊窗口，命中時間−拍點=時間差，|dt|≤perfectMs 完美、≤goodMs*0.85 不錯、其餘視為搶拍（+5 不計連擊）
difficulty_params: 節拍間隔 1800/1250/950ms；時機窗 ±520/450/400ms（完美 ±220/180/150ms）；鼓面半徑 80/66/56px@640；鼓位 2/2/3（hard 含中央跨中線）
metrics: score、hits/misses→節奏準確率、avg_reaction_ms=平均拍點時間差、left/center/right_hits=左右分布、結算頁另示最高連擊與左右交替率
compensation_rules: 沿用全域 usePoseMonitor（聳肩/軀幹前傾/側彎，畫面＋語音溫和提示）
contraindications: 上肢急性疼痛或不穩定骨折、光敏性癲癇病史者建議治療師評估後使用（視覺為平滑霓虹呼吸，無 >3Hz 閃爍，仍列出供把關）
safety: 坐姿為預設建議（文案載明「坐著玩就可以」）；單手偵測，無強制雙手要求，患側不可及時可由健側代打不受懲罰；失誤僅灰色提示無懲罰演出；震動/音量沿用平台開關
fun_layer: calm 星夜暗底＋霓虹三色鼓面（青/紫/洋紅）＋光軌落音符＋節拍呼吸背景（≤1.1Hz）＋命中霓虹 burst/彈跳字/微震＋combo 里程碑光效與 hit-stop（僅 ×10）
session_structure: 單回合 60 秒；建議處方每週 3–5 次，回合間休息 1–2 分鐘
```
